# 자동 게시 설정 가이드

## 무료 자동 게시 가능 / 불가

| 플랫폼 | 자동 게시 | 비용 | 필요한 것 |
|--------|-----------|------|-----------|
| **Discord** | ✅ | 무료 | Webhook URL |
| **Reddit** | ✅ | 무료 | API 앱 + 계정 |
| X | ❌ (유료 API) | 월 $100~ | — |
| Threads | ❌ | — | 수동 |
| Google Play | ❌ | — | 수동 |
| YouTube/TikTok | ❌ | — | 수동 (영상) |

## Discord Webhook

1. Discord 서버 → 채널 → 연동 → 웹후크 → URL 복사
2. `.env`에 `DISCORD_WEBHOOK_URL=...`
3. GitHub Secret에도 등록

## Reddit API

1. https://www.reddit.com/prefs/apps → script 앱 생성
2. `.env`에 CLIENT_ID, SECRET, USERNAME, PASSWORD
3. `config.json`에 `account_created_date` 설정

## 실행

```bash
python run_free.py --publish --approve
```
